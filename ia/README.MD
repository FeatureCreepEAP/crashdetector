
# Preparación de Datos para el Agente de IA

Esto es para crear los datos para un Agente de IA de Baidu:
https://agents.baidu.com/

## Pasos

Ejecutar:
./../generar_codigo_para_agente.sh

Esto generará:
./ia/codigo_para_agente.txt

## Archivos

codigo_para_agente.txt  
Contiene todo el código que se enviará al Agente de IA.

info_para_ia.txt  
Contiene información básica para el prompt del agente.  
Dado que la IA de Baidu funciona principalmente en chino, este archivo está escrito en chino.
